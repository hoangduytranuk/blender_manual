Khôi Phục Lại

Scene orientation tools can be used for orienting object to bundles.

Đặt Gốc Tọa Độ

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Set Origin

Transform camera in a way which makes active track to be moved to a scene origin. Only translation is applied to the camera.

Đặt Sàn

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Set Floor

Use selected three markers to define a floor. Camera will be transformed in a way which makes the selected markers to be flat (have Z = 0).

Đặt Tường

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Set Wall

Similar to the floor orientation, but defines a wall (selected tracks are placed onto the XZ plane).

Đặt Trục X/Y

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Set X/Y Axis

Transform camera in a way which makes active track to become on X or Y axis. No translation is applied, meaning scene origin which was specified before will be preserved.

Đặt Tỷ Lệ

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Set Scale

Scale camera or tracking object in a way which makes distance between two selected tracks match the given value in Distance.

Áp Dụng Tỷ Lệ vào Giải Nghiệm

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Apply Solution Scale

Similar to Đặt Tỷ Lệ, but actually modifies the tracking data.

Kết Nối Vật Thể Rỗng với Giám Sát

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Reconstruction ‣ Link Empty to Track

Creates new empty in 3D Viewport and appends constraint which parts it to the active track.

Các Dấu Mốc 3D thành Khung Lưới

Tham Chiếu

Chế Độ

Giám Sát

Trình Đơn

Chọn Lựa (Select) ‣ Nới Rộng/Thu Nhỏ (More/Less)

Creates a mesh which vertices matches positions of reconstructed tracks. It is required to have motion solved first before using this operator. Only tracks from the current tracking object will be used. The intention of this operator is to give a nice starting point for a manual mesh reconstruction.
