Ràng Buộc Sao Chép sự Xoay Chiều -- Copy Rotation Constraint

The Copy Rotation constraint forces its owner to match the rotation of its target.

Tùy Chọn -- Options

Mục Tiêu -- Target

Data ID used to select the constraints target, and is not functional (red state) when it has none. See common constraint properties for more information.

Theo Thứ Tự

Allows specifying which Euler order to use during the copy operation. Defaults to the order of the owner.

Axis -- Trục

Những nút bấm này điều khiển trục nào là cái bị ràng buộc.

Đảo Nghịch -- Invert

Invert their respective corresponding axis coordinates.

Hòa Trộn -- Mix

Specifies how the new rotation is combined with the existing rotation.

Thay Thế -- Replace

The new axis values replace existing values.

Cộng Thêm -- Add

The new axis values are added to the existing values.

Trước Nguyên Bản

The new rotation is added before the existing rotation, as if it was applied to a parent of the constraint owner.

Sau Nguyên Bản

The new rotation is added after the existing rotation, as if it was applied to a child of the constraint owner.

Dịch Chuyển (Lỗi Thời)

This replicates the behavior of the original Offset checkbox. It was intended to be similar to the Before Original behavior, but does not work correctly with multiple axis rotations, and is thus deprecated.

Mục Tiêu/Chủ Nhân

Standard conversion between spaces. See common constraint properties for more information.

Tác Động/Ảnh Hưởng -- Influence

Controls the percentage of affect the constraint has on the object. See common constraint properties for more information.
