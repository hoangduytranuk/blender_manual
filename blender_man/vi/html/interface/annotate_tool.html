Chú Thích Công Cụ

The annotation tool is available in multiple editors. It can be used to add notes to e.g. 3D objects or node setups.

The annotation tool can be activated in the Toolbar and has following sub-tools:

Chú Thích

Vẽ chú thích tự do bằng tay.

Chú Thích bằng Đường Thẳng

Click and drag to create a line. Optionally, you can select the arrow style for the start and end of the line.

Chú Thích bằng Đa Giác

Click multiple times to create multiple connected lines. The current polygon is finished when Esc is pressed.

Tẩy xóa Chú Thích

Click and drag to remove lines. The eraser has a Radius setting found in Tool Settings ‣ Eraser.

Sắp Đặt

Sắp Đặt Chung/thông thường/phổ biến/chung

Màu Sắc

Adjust the color of existing and new strokes.

Chú Thích -- Annotations

A pop-over menu, showing the name of the current layer, to access the Annotation Layers.

Sắp Xếp

Determines where the annotations are drawn.

Con Trỏ 3D

Draw annotations on an imaginary plane that goes through the Con Trỏ 3D and is aligned to your view.

Góc Nhìn

Draw annotations in screen space rather than 3D space; meaning the annotations will stay on the same position in the screen, even when the view moves or rotates.

Bề Mặt

Draw annotations on the surface of the object under the mouse.

Hình Ảnh

Draw annotations in same space as the image/preview/nodes meaning as you zoom in or out the annotations stay the same size relative to the zoom factor.

Ổn Định Hóa Nét Vẽ

Helps to reduce jitter of the strokes while drawing by delaying and correcting the location of points.

Bán Kính

Minimum distance from the last point before the stroke continues.

Hệ Số

A smooth factor, where higher values result in smoother strokes but the drawing sensation feels like as if you were pulling the stroke.

Chú Thích bằng Đường Thẳng

Phong Cách cho Đầu, Cuối

The decoration to use at the beginning or end of the line segment. This can be used for example to create arrows to point out specific details in a scene.

Tầng Lớp Chú Thích

When the annotation tool is enabled, the settings for managing multiple layers can be found in the Sidebar ‣ View ‣ Annotations panel.

Độ Đục

Adjusts the opacity of existing and new strokes.

Độ Dày

Adjusts the thickness of existing and new strokes.

Bóc Vỏ Hành

Shows a ghosted image of strokes made in frames before and after the current frame. Onion skinning only works in the 3D Viewport and Sequencer. See the Grease Pencil documentation for an explanation of Onion Skinning.
