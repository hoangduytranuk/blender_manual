Cài đặt trên Windows -- Installation on Windows¶

Hướng dẫn này bao gồm các chủ đề sau:

  1. Cài đặt Python -- Installing Python (được sử dụng để "chuyển đổi" các tập tin nguồn sang dạng HTML)
  2. Cài Đặt SVN và Tải Kho Lưu Trữ về Máy -- Installing SVN and Downloading the Repository
  3. Thiết Lập Môi Trường Biên Tập -- Setting up the Build Environment

Cài Đặt Python -- Installing Python¶

  1. Tải xuống máy Gói cài đặt Python -- dành cho Windows. Trong hướng dẫn này phiên bản 3.6.x sẽ được sử dụng.

  2. Cài đặt Python bằng trình hướng dẫn cài đặt. Xin vui lòng đảm bảo rằng bạn đã bật tùy chọn Thêm Python vào PATH -- Add Python to PATH:

    Tùy chọn phải được bật để bạn có thể biên dịch bản hướng dẫn sử dụng bằng tập lệnh make.

    Tất cả các cài đặt khác có thể vẫn để như sắp đặt mặc định.

Cài Đặt SVN và Tải về Máy Kho Lưu Trữ -- Installing SVN and Downloading the Repository¶

Trong hướng dẫn này, chúng tôi sẽ sử dụng TortoiseSVN, mặc dù bất kỳ trình khách ứng dụng Subversion nào cũng có thể sử dụng được.

  1. Tải xuống máy bản TortoiseSVN -- dành cho Windows.
  2. Cài đặt TortoiseSVN bằng trình hướng dẫn cài đặt. Khi chọn các tính năng sẽ được cài đặt, bạn nên cho bật công cụ trình khách dòng lệnh -- command line client tools để cho phép mình truy cập vào SVN từ dòng lệnh (thực hiện việc này không có gì hại cả, và ngược lại, có thể rất hữu ích nếu bạn gặp phải bất kỳ rắc rối nào).
  3. Sau khi cài đặt xong, hãy tạo một thư mục mới để chứa tất cả mọi thứ liên quan đến Bản Hướng Dẫn Sử Dụng Blender. Trong hướng dẫn này, chúng ta sẽ sử dụng C:\blender_docs.
  4. Mở thư mục mới, nhấp chuột phải và chọn SVN Checkout ... (Kiểm Xuất SVN) từ trình đơn ngữ cảnh.
  5. Trong ô URL of repository (Địa chỉ của kho lưu trữ), làm ơn điền: https://svn.blender.org/svnroot/bf-manual/trunk/blender_docs.
  6. Trong ô Checkout directory (Thư mục kiểm xuất), làm ơn điền: C:\blender_docs.
  7. Nhấp vào OK (Chấp Thuận) -- kho lưu trữ hiện sẽ được tải xuống, có thể mất vài phút, tùy thuộc vào chất lượng đường kết nối Internet của bạn.

Thiết Lập Môi Trường Biên Dịch -- Setting up the Build Environment¶

  • Mở một Nhắc Lệnh. (Chạy trong chế độ Người Quản Trị Hệ Thống -- Run as Administrator)

  • Bước vào thư mục blender_docs vừa được kiến tạo và bổ sung bởi việc kiểm xuất (checkout) SVN:

    cd C:\blender_docs

  • Bên trong thư mục đó có một tập tin với cái tên requirements.txt, và trong đó có chứa một danh sách tất cả các phần mềm phụ thuộc mà chúng ta cần phải có. Để cài đặt các phụ thuộc này, chúng ta có thể sử dụng lệnh pip như sau:

    pip install -r requirements.txt

  • Nếu mọi việc xảy ra trôi chảy thì bạn sẽ thấy thông báo sau khi nó hoàn thành:

    Successfully installed Jinja2 MarkupSafe Pygments Sphinx docutils sphinx-rtd-theme Cleaning up...

Trong khi thiết lập, một số cảnh báo có thể xuất hiện, nhưng đừng lo lắng về chúng. Đương nhiên, nếu có bất kỳ lỗi nào xảy ra, chúng có thể gây ra một số rắc rối.

Ghi chú

Thỉnh thoảng, bạn có thể muốn đảm bảo là các thư viện phụ thuộc của bạn được cập nhật với những thay đổi gần đây nhất, bằng cách sử dụng:

pip install -r requirements.txt --upgrade

Tiếp tục với bước tiếp theo: Biên Dịch.
