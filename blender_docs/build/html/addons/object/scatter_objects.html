Scatter Objects¶

Reference

Category

Object

Description

Distribute object instances on another object.

Location

3D View ‣ Search

File

object_scatter folder

Author

Jacques Lucke

Installation¶

  • This add-on is bundled with Blender.

  • Open Blender and go to Preferences then the Add-ons tab.

  • Click Object then Scatter Objects to enable the script.
